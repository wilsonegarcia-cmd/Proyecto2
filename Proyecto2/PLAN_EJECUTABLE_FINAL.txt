🔵 FASE 3: FIRMWARE ESP32 (1-2 HORAS)

La parte más larga pero sigue siendo paso a paso.

PASO 1: Instalar ESP-IDF (15 min)

  Ve a: https://github.com/espressif/esp-idf/releases
  
  Busca: esp-idf-tools-setup-x.x.exe (última versión)
  
  Descarga y ejecuta
  
  Sigue el wizard (siguiente, siguiente, siguiente)
  
  Al terminar, cierra.
  
  Abre CMD nueva y verifica:
    idf.py --version
  
  Debería mostrar: ESP-IDF v5.1.x (o superior)

PASO 2: Crear carpeta para el código (1 min)

  Crea carpeta (si no existe):
    c:\Users\Eduardo Garcia\Desktop\Proyecto2\esp32\main\

PASO 3: Copiar código ESP32 (3 min)

  Abre archivo: CODIGO_ESP32.txt
  Copia TODO
  Crea archivo: esp32/main/main.c
  Pega TODO el contenido

PASO 4: Configurar WiFi (2 min)

  Abre: esp32/main/main.c (con Notepad o editor)
  
  Busca (línea ~25):
    #define WIFI_SSID "CAMBIAR_A_TU_RED"
    #define WIFI_PASSWORD "CAMBIAR_A_TU_PASS"
    #define SERVER_IP "192.168.1.100"
  
  Cambia a:
    #define WIFI_SSID "TuRedWiFi"           ← Tu WiFi
    #define WIFI_PASSWORD "tuContraseña"     ← Tu contraseña
    #define SERVER_IP "192.168.1.50"         ← Tu IP (mira abajo)
  
  ¿CÓMO OBTENER TU IP?
    Abre CMD
    Escribe: ipconfig
    Busca: IPv4 Address: 192.168.X.X ← ESE NÚMERO
  
  Guarda el archivo

PASO 5: Conectar ESP32 por USB (1 min)

  Conecta ESP32 a la computadora por cable USB
  
  Windows detectará un puerto COM (COM3, COM4, etc.)
  
  Para verificar:
    Device Manager → Ports → COM?

PASO 6: Compilar (10 min)

  Abre CMD
  
  cd c:\Users\Eduardo Garcia\Desktop\Proyecto2\esp32
  
  idf.py build
  
  Espera 10 minutos...
  
  Cuando termine, debería decir: "✓ Project built successfully"

PASO 7: Subir a ESP32 (5 min)

  En la misma terminal:
  
  idf.py -p COM3 flash
  
  (Cambiar COM3 por tu puerto real)
  
  Verás:
    Hard resetting esp32...
    Leaving...
  
  Espera 5-10 segundos

PASO 8: Verificar (1 min)

  idf.py -p COM3 monitor
  
  Verás logs como:
    I (XXX) BT: Bluetooth Initialized
    I (XXX) WIFI: WiFi Initialized
    I (XXX) WIFI: Connected to WiFi
  
  Si ves "Connected to WiFi" = ¡Funcionando!
  
  Presiona Ctrl+] para salir

✅ FASE 3 COMPLETADA

El ESP32 tiene firmware y está funcionando.
