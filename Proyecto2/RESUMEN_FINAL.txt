💡 CONSEJOS IMPORTANTES

✓ SERVIDOR
  • Ejecutar primero el servidor antes de usar ESP32/Android
  • Verificar que Puerto 5000 está disponible
  • Si error "Port already in use", cambiar puerto en app.py

✓ ANDROID
  • Permisos se solicitan en runtime (Android 6+)
  • Usar dispositivo real para Bluetooth (emulador tiene limitaciones)
  • WiFi del Android puede ser diferente a la del ESP32

✓ ESP32
  • Escribir IP local correcta del servidor (no localhost)
  • Token debe terminar con \n (salto de línea)
  • Si desconexión, revisar conexión Wi-Fi en monitor
